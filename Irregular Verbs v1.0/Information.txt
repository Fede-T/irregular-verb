To add a verb, type it as follows:

Infinitive_verb Past_simple Past_participle

a "_". For example:

"must hail_to had_to" --> the program will take as "hail to" and "had to".

If a verb has two alternatives "/" should be added, for example:

"be was/were been" --> the program will take as correct: was/were - were/was - was - were.

ES

Para agregar o quitar verbos, ingresar al archivo de texto "verb list" y quitar
los verbos que quiera.

Para agregar un verbo se debe escribir de la siguiente manera:

Infinitive_verb Past_simple Past_participle

si un verbo tiene un espacio como por ejemplo "had to" no se debe agregar un espacio, se debe agregar
un "_". Por ejemplo:

"must hail_to had_to" --> el programa tomará como "hail to" y "had to"

Si un verbo tiene dos alternativas se debe agregar "/", por ejemplo:

"be was/were been" --> el programa tomará como correcto: was/were - were/was - was - were

Este programa es totalmente gratuito.
